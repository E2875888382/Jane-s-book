# webpack-vue-express
- 项目名称：视频社交社区
  - 已经实现的功能
    - 登录（邮箱作为账号）
    - 注册（发送邮箱验证码）
    - 查看个人信息
    - 修改个人信息
    - 查看账号安全状况
    - 读好友消息
    - 管理好友（添加好友，删除好友）
- 技术栈：
  - vue.js
  - webpack
  - express
  - vuex
- 运行方式
  - 1.安装依赖
    - ```npm i  ```
  - 2.在app.js下开启后台服务器（app.js），端口号：8000
    - ```node app.js ```
  - 3.在根目录下开启webpack-dev-server的服务器（打包前端资源），端口号：3000
    - ```npm run dev ```
- webpack编译完成会自动打开浏览器（loaclhost:3000/）
